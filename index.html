迁移Hexo

这个博客是之前花了2天的时间搭好的,但是期间遇到了许多问题,今天趁着没什么事情,在MBA上,再搭建一次.

第一步:环境搭建

你需要以下环境

  • git(点击Xcode即送)
  • Nodejs 点击我进入官网下载 安装的时候版本为v6.9.2

第二部:主角登场

博客的话,我使用的Hexo

$ npm install hexo-cli -g 在终端输入后,等待左边的进度条跑完

然后输入密码,等待左边的进度条走完.

ok成功

到此为止,hexo的所需环境就搭建完成了

迁移动作

将xxx.github.io文件拷到新电脑中

然后cd到该文件中

$ cd ~/heTaoEvo.github.io

然后发现了一个问题,
-bash: ~/Blog/heTaoEvo.github.io: is a directory

看样子需要重新创建一个新的文件夹,然后将内容拷贝过去

$ hexo init 你的用户名.github.io

然后将之前的数据拷贝进去.输入

npm install hexo-deployer-git --save

安装自动部署工具,等待进度条.最后

hexo clean && hexo g && hexo d

2016年终总结

2016年是变化的一年,从广东,到成都;从刚刚毕业的学生,到初入职场的新人.之前也从来没有写过年终总结,趁着这几天刚刚搭好的博客,来好好梳理下今年吧.

工作

  • 首先是换了工作吧,从深圳回到了成都,无论是地点,还是公司,都换了(这tm不是废话么?),从日常的时间也有非常的大的变化,从之前的两点一线变成三点两线.
  • 工作的兴趣比在广东好多了,在广东的时候,就感觉像混吃等死,但是回到了成都,基本都想学习,学习更多的东西,或许这就是成长,或许这就是改变.
  • 对iOS的兴趣也正是导致第二条的原因吧,确实现在市场竞争很大,从事这个行业的人也很大,但是,我依旧坚信,工作岗位始终是有的,只有你比别人强,那么那个位置就是你的.所以,加油努力吧!
  • 对于其他的语言.这也算工作的一部分吧,由于之前欠下的债,导致iOS确实不是很厉害,也没有利用周末好好学习,都去跟麻将杂志社活动了,确实应该检讨.(讲道理,单双休确实感觉时间不够啊!
  • 检讨上一条,周末戒掉WOW(反正买不起点卡,也开不出橙),利用周末时间多看看Python吧(廖雪峰,我来了).

总之,工作也许有点不尽人意,但是总体来说,感觉不是不可以打个80分.

生活

首先,我对现阶段的生活十分满意,虽然简单,但是充实.从上午到下午,每天都是不知不觉就过完了

  • 工作日十分充实,但是感觉每天步行回家的一个小时锻炼有限(肚皮越来越来大了).
  • 周末回到家,跟爹妈一起,说道爹妈,之前去广东,他们也没有反对,自己也跟白眼狼一样,没有任何不适,也几乎没用主动打过电话,他们也没说什么,但是回到成都了,能明显感觉他们变得高兴了,不管之后怎样,这个回来的确定就现在来讲,是没有错的.
  • 爹妈一条肯定说不完的,跟别人不同,我老妈总是问我钱够不够,无论学生还是工作,老爸总是让我去买新衣服,不去还要生气,感觉自己也不是很争气,一直让他们很担心,前段时间,买房子的风波也确实让自己感受到压力,所以,还是那一句,努力吧.
  • 生活是不能缺少朋友的,周末的八卦小组,虽然人少,但是感情好,带来了许多欢乐,但是,希望社长的离开,不能影响了我们!
  • 要说最近喜欢玩的游戏,应该就是赛车了吧,借着黑五,补全了装备,G29,虽然周末时间不多,但是几把的时间足以让我身临其境
  • 2016的新兴趣应该就是F1了吧,伴随我几乎全年的时光,每每看到有人说F1不行了,我就会说Whatever,这样挺好.
  • dota2的话基本完全不玩了,看看直播当平时的消遣吧.

生活是由各个琐事拼接而成,但是正是这些琐事,才让我们对生活有所期望
