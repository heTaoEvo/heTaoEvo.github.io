迁移Hexo

这个博客是之前花了2天的时间搭好的,但是期间遇到了许多问题,今天趁着没什么事情,在MBA上,再搭建一次.

第一步:环境搭建

你需要以下环境

  • git(点击Xcode即送)
  • Nodejs 点击我进入官网下载 安装的时候版本为v6.9.2

第二部:主角登场

博客的话,我使用的Hexo

$ npm install hexo-cli -g 在终端输入后,等待左边的进度条跑完

然后输入密码,等待左边的进度条走完.

ok成功

到此为止,hexo的所需环境就搭建完成了

迁移动作

将xxx.github.io文件拷到新电脑中

然后cd到该文件中

$ cd ~/heTaoEvo.github.io

然后发现了一个问题,
-bash: ~/Blog/heTaoEvo.github.io: is a directory

看样子需要重新创建一个新的文件夹,然后将内容拷贝过去

$ hexo init 你的用户名.github.io

然后将之前的数据拷贝进去.输入

npm install hexo-deployer-git --save

安装自动部署工具,等待进度条.最后

hexo clean && hexo g && hexo d
